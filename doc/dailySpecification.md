### 二、   考核基本要求
**针对组长**

1 ) 对团队有一定贡献。（优秀）

2 ) 让组员有成长。（良好）

3 ) 把交代的事情做好。（合格）

**针对所有人**
 
1 ) 把已安排的工作做好，并能把领导没有吩咐的事情都做好，为领导解忧。（优秀 A）

2 ) 安排的工作都能做好。（合格 B/C）

3 ) 安排的工作都做不好。（不合格 D）
